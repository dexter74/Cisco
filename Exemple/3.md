#### III. Router0
Configuration de l'adresse IP `192.168.1.1/24` et activation de l'interface `GigabitEthernet 0/0/0`
```
enable
configure terminal
interface GigabitEthernet 0/0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
```
