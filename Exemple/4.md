### II. Switch
#### A. Base
```
enable
configure terminal
hostname Switch
```

#### B. Configuration
```
vlan 10
name Voice

vlan 20
name Data

interface vlan 10
ip address 192.168.1.1 255.255.255.0
no shutdown

interface vlan 20
ip address 192.168.2.1 255.255.255.0
no shutdown

exit
```
