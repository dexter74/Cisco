### Présentation
Un routeur fournissant des services DHCP avec séparation avec le VLAN.

L'interface `FA 0/0`du routeur sera initailisé uniquement.

Le `VLAN 10` se nommera `CLIENT`.

Le `VLAN 20` se nommera `MANAGEMENT`.

----------------------------------------------------------------------------------------------------------------------------------
### Equipements
```
- Routeur : 2811
- Switch  : 2960
- PC      : 2
```
<br />

----------------------------------------------------------------------------------------------------------------------------------
### Switch
#### Mode privilège
```bash
enable
configure terminal
```

----------------------------------------------------------------------------------------------------------------------------------
### Routeur
#### Mode privilège
```bash
enable
configure terminal
```

### Création des VLANS
```
vlan 10
name VOICE
vlan 20
name PC
exit
```

### Configuration du TRUNK
```
interface fa 0/1
switchport mode trunk
exit
```

### Configuration des ports
```
interface range fa 0/2-24
switchport mode access
switchport access vlan 20
switchport voice vlan 10
exit
```
